# AdsDistribution  
Приложение для быстрой рассылки постов в несколько разных групп  

## Инструкция по использованию  
Для работы необходимо создать config.json в папке с исполняемым файлом со следующим содержанием:  
```
{
  "Login": "<Логин>",
  "Password": "<Пароль>",
  "AppId": 7049623,
  "Message": "<Сообщение>",
  "GroupsIds": [
    <id групп через запятую>
  ]
}
```
